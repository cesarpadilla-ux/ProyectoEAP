Protocolo de Inicio

01 / Alimentación
Conecte el dispositivo a una toma de corriente estable.
02 / Encendido
Active los interruptores de Batería y Alimentación Principal.
03 / Periféricos
Conecte monitor HDMI, teclado y mouse.
04 / Sistema
Espere la carga completa de la interfaz de usuario.
05 / Red
Establezca conexión vía WiFi o Ethernet (LAN).
06 / Terminal
Abra la terminal de comandos.
07 / Directorio
Ejecute: cd Desktop/EAP
08 / Sensores
Coloque electrodos ECG y sensores PPG al paciente.
09 / Ejecución
Ejecute: python3 eap.py
10 / Captura
Adquiera datos por >15 segundos. Pulse "Detener" al finalizar.
11 / Verificación
Confirme mensaje de "Transferencia Exitosa".
12 / Visualización
Refresque esta página para cargar las nuevas señales.
